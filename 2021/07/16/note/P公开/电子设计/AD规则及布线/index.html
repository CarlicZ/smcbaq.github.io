AD规则及使用技巧

[toc]

前言

最近学AD画板子，对AD中的设计规则和布线技巧不怎么懂，所以记录在这个博客上，如果有新的技巧持续更新。

AD规则介绍

规则 内容 备注
电气规则（Electrical）
Clearance 间距设置 通常忽略同一封装内间距
Short-Circuit 短路设置 当然不允许
Un-Routed Net 不连接的线
Un-Connected Pin 不走线的引脚
Modified Polygon 允许修改
走线（Routing）
Width 线宽 根据需求每次设置
Routing Corners 走线角度 一般45度
Routing Via Style 过孔设置 看需求
DiffPairsRouting 差分对设置 两条线间隔的最值
SMT（表贴焊盘规则）
SMD To Corner 焊盘与导线拐角最小间距
SMD To Plane 焊盘与电源层过孔最小间距
SMD Neck Down SMD焊盘颈缩率规则
Mask（阻焊层规则）
Solder Mask Expansion 阻焊层收缩量规则
Paste Mask Expansion 助焊层收缩量规则
Plane（电源层规则）
PlaneConnect 电源层连接类型
PlaneClearance 电源层安全间距
PolygonConnect 焊盘与覆铜连接类型

AD各层含义

层名 中文 含义
TOP LAYER 顶层布线层 顶层铜箔走线，单面板则没有该层
BOMTTOM LAYER 底层布线层 底层铜箔走线
TOP/BOTTOM SOLDER 顶层/底层阻焊绿油层 顶层/底层敷设阻焊绿油，保持绝缘，焊盘、过孔及本层非电气走线处阻焊绿油开窗
TOP/BOTTOM PASTE 顶层/底层锡膏层 该层一般用于贴片元件的SMT回流焊过程时上锡膏
TOP/BOTTOM OVERLAY 顶层/底层丝印层 各种丝印标识
MECHANICAL LAYERS 机械层 设计为PCB机械外形，默认1为外形层。其它可作为机械尺寸标注或者特殊用
KEEPOUT LAYER 禁止布线层 可做PCB机械外形，同时有KEEPOUT和MECHANICAL，一般以MECHANICAL为准。
MIDLAYERS 中间信号层 多用于多层板
INTERNAL PLANES 内电层 用于多层板
MULTI LAYER 通孔层 通孔焊盘层
DRILL GUIDE 钻孔定位层 焊盘及过孔的钻孔的中心定位坐标层
DRILL DRAWING 钻孔描述层 焊盘及过孔的钻孔孔径尺寸描述层
